# Kilo

一千行 C 代码实现的带有搜索功能和语法高亮的终端文本编辑器，无任何外部依赖

# 教程

https://viewsourcecode.org/snaptoken/kilo/

# 按键

```
Ctrl-X 退出
Ctrl-S 保存
Ctrl-F 查找（ESC 取消，方向键在结果之间跳转，Enter 留在当前查找结果）
```
